- **`vf_pelorus_denoise_vulkan`** (`ffmpeg-patches/files/vf_pelorus_denoise_vulkan.c`): an edge-preserving spatio-temporal denoise filter — the biggest BD-rate pre-encode lever. Single Vulkan compute pass over a causal window (current + *N* previous frames in VRAM, no motion compensation): an NLM-lite joint bilateral (3×3-patch-SSD range weight that refuses to average across edges) blended with per-pixel-gated temporal averaging (a delta above `tcut` breaks the walk so motion can't ghost). AVOptions: `sigma`, `sigmac`, `sigmat`, `strength`, `strengthc`, `blend`, `tdecay`, `tcut`, `patch`, `prev`, `protect`, `planes`, `meta`. With `meta=1` it emits the pre-reserved `PEL_SEC_DENOISE` interop section (residual energy, applied strength, noise-σ estimate, PSNR-vs-input). Host-side param contract `pelorus/denoise.h` (`pel_denoise_params_default`/`_validate`); reference shader `libpelorus/shaders/pelorus_denoise.comp` kept in lockstep. Shipped as `ffmpeg-patches/0003-add-vf_pelorus_denoise_vulkan.patch`. ADR-0112, ADR-0111.
- **Param contract** (`libpelorus/include/pelorus/denoise.h`, `libpelorus/src/denoise_params.c`): `PelorusDenoiseParams` + defaults/validation, the vmafx-autotune-facing view of the denoise knobs. ADR-0112.
