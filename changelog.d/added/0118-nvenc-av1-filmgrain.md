- **NVENC hardware AV1 film grain** (`ffmpeg-patches/files/nvenc-pelorus-film-grain.patch`, `-pelorus_film_grain 1`, `av1_nvenc` only): the hardware-AV1 leg of the FGS round-trip (ADR-0118). Stock `av1_nvenc` never enables NVENC's AV1 film-grain synthesis, so the estimate `vf_pelorus_grain_estimate_vulkan` (ADR-0115) emits is dropped and the grain is coded as residual. This patch carries that estimate into NVENC's AV1 film-grain config: at init it sets `NV_ENC_CONFIG_AV1::enableFilmGrainParams` and points `filmGrainParams` at a persistent `NV_ENC_FILM_GRAIN_PARAMS_AV1`; per frame it refills that struct from the frame's grain params (the native `AV_FRAME_DATA_FILM_GRAIN_PARAMS`/`AV_FILM_GRAIN_PARAMS_AV1` channel preferred, the `PEL_SEC_FILMGRAIN` interop section parsed inline as fallback) and raises the AV1 pic-params `filmGrainParamsUpdate` flag (time-varying model). The `AVFilmGrainAOMParams` set maps field-for-field onto the NVENC struct (numbers shifted to NVENC's "minus N"/"plus 128" conventions, point and AR-coefficient arrays copied element-wise, all writes clamped). With the H.274 BSF (patch 0010) for HEVC and native side data for AV1 software encoders, this completes the FGS round-trip on NVIDIA hardware AV1.
