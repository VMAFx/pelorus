- **`vf_pelorus_grain_estimate_vulkan`** (`ffmpeg-patches/files/vf_pelorus_grain_estimate_vulkan.c`): a film-grain-synthesis (FGS) parameter **estimator** — the dual of the denoise lever. A pass-through Vulkan compute pass (modelled on `vf_pelorus_analyze_vulkan`) measures, per luma intensity band, the high-frequency residual variance (luma minus a 3×3 box low-pass) over locally-flat neighbourhoods (an edge gate excludes structure), plus a global lag-1 spatial-correlation accumulator as a coarse AR seed. The per-band RMS residual maps to the AV1 AOM piecewise scaling function `y_points[value, scaling]`; the lag-1 proxy seeds `ar_coeffs_y`. AVOptions: `edge`, `strength`, `model` (`aom`/`h274`), `native`. The filter emits BOTH the pre-reserved `PEL_SEC_FILMGRAIN` interop section (codec-neutral intent + AV1 params + H.274 mode scalars + `grain_model` tag; no ABI bump) AND a native `AV_FRAME_DATA_FILM_GRAIN_PARAMS` (`AV_FILM_GRAIN_PARAMS_AV1`) so a downstream FFmpeg AV1 encoder honours the estimate with no Pelorus-specific BSF. Reference shader `libpelorus/shaders/pelorus_grain_estimate.comp` kept in lockstep. The AV1-OBU/H.274-SEI bitstream filter, the full per-lag AR fit, and explicit chroma estimation are documented follow-ups. Shipped as `ffmpeg-patches/0006-add-vf_pelorus_grain_estimate_vulkan.patch`. ADR-0115.
