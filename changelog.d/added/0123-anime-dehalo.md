- **`vf_pelorus_dehalo_vulkan`** (`ffmpeg-patches/files/vf_pelorus_dehalo_vulkan.c`): anime / 2D **dehalo + dering** — a single-pass zero-copy Vulkan compute port of HAvsFunc `DeHalo_alpha` + `FineDehalo` (`FF_VK_REP_FLOAT` UNORM, bit-depth-agnostic, luma only; chroma passes through). A strong box blur is the halo-free target; a `lowsens`/`highsens` sensitivity mask from the removed local contrast drives a remove-only asymmetric pull toward the blur (`darkstr` for dark halos, `brightstr` for bright); a `FineDehalo` Sobel edge mask dilated by `ring` confines the pull to the halo band next to line-art so the drawing and open gradients are protected. AVOptions: `blur`, `darkstr`, `brightstr`, `lowsens`, `highsens`, `edge`, `ring`, `planes`. A pure pixel transform — it does not link libpelorus and emits no interop side data. Foundation of the planned `tune=anime` pipeline. Reference shader `libpelorus/shaders/pelorus_dehalo.comp` kept in lockstep. Compile-verified and glslang-clean; the on-content tuning of the defaults and the SSIMULACRA2 / edge-region VMAF-NEG / CAMBI proof against a clean anime ground truth are documented follow-ups (no BD-rate/quality number claimed yet).
