- **`vf_pelorus_aa_vulkan`** (`ffmpeg-patches/files/vf_pelorus_aa_vulkan.c`): anime warp anti-aliasing + optional line-darkening — a single-pass, zero-copy Vulkan compute (`FF_VK_REP_FLOAT` UNORM, bit-depth-agnostic) port of awarpsharp2 + FastLineDarken, and the second stage of the anime `tune` chain after `vf_pelorus_dehalo_vulkan` (ADR-0123). It builds a blurred Sobel edge-strength map and *warps* each luma pixel toward the nearest edge by the gradient of that map (bilinear resample at the displaced position), pulling stair-stepped "jaggie" samples onto the line — anti-aliasing + thinning with no sharpen ringing — then optionally deepens the dark side of real edges (Sobel-gated) so lines stay crisp. Luma only; chroma passes through. A pure transform: it emits no side data and does **not** link `libpelorus` (deps-only registration). AVOptions: `blur` (2), `depth` (8.0), `thresh` (0.5), `darkstr` (0.0 = off), `edge` (0.08), `planes` (0x1 = luma). Reference shader `libpelorus/shaders/pelorus_aa.comp` kept in lockstep. Honest caveat: the port is compile-verified + glslang-clean only — the defaults are unproven on content, and the SSIMULACRA2 / edge-region VMAF-NEG proof against clean anime ground truth is the documented follow-up; no quality number claimed. Shipped as patch 0012. (ADR-0124)
