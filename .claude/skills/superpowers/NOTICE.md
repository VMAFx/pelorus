# Vendored: obra/superpowers

[obra/superpowers](https://github.com/obra/superpowers) by Jesse Vincent,
MIT-licensed (see `LICENSE`). They are general-purpose engineering-process skills
(verification-before-completion, systematic-debugging, test-driven-development,
writing/executing-plans, code-review, git-worktrees, …) reused here per the
project rule to leverage proven prior art rather than reinvent.

Update: re-pull from the upstream repo (or install the Claude Code plugin) rather
than hand-editing these files; Pelorus-specific skills live one level up in
